流水线接入指南-JAVA经典型

注意：
现之家云为推进应用标准化，将通过邮件/应用列表/上线单等多种渠道提醒您及时审查应用非标准化原因，且应用上线单审批系统会自动检测发起上线审批的应用是否标准化，是则通过，否则反之。故请您务必及时审查应用非标准化原因，以免影响上线单审批效率。

  • 若0/1级应用未关联SLA监控：建议您查看0级应用关联URL监控操作说明,并及时关联SLA监控
  • 若镜像非标准化:建议您查看业务方自定义镜像（工具镜像）管理文档
  • 若日志采集非标准化:建议您查看 Autolog日志采集并投递鹰眼日志操作文档

如仍有任何相关问题，您均可本人或者安排对接人咨询之家云运营团队：陈超、王成刚、魏大志、刘毅

1. 从之家云应用页面新建应用


注意：

  1. 应用名称必须字母开头，支持小写字母、数字、横划线，但必须包含字母。名称全公司唯一。 最多25位。一旦设置不可修改
  2. 应用等级解释：
    0级-核心应用（核心流量服务）；1级-重要应用（支持0级的相关应用）；
    2级-普通应用（公司内部相关应用）；3级-测试应用（测试或调试使用的相关应用）
    目前之家云只支持创建1级/2级/3级应用，升级至0级应用需要提交应用升级工单，具体详见：应用等级升降级帮助文档，且业务方应用上线时需确认0/1级应用是否已配置SLA监控，若未关联SLA监控：建议您查看0级应用关联URL监控操作说明，并及时关联SLA监控，否则会影响应用上线审批效率。
